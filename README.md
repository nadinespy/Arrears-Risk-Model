# Arrears Risk Model

Productionised pipeline for predicting household arrears risk, built around a prior analytical project on Lewisham council data (17,831 households).

This repository wraps an `src/` package, tests, configuration, schemas, and CI around the original notebook analysis. The notebooks are preserved as the analytical reference; the production code lives in `src/arrears_risk_model/`.

## Repository structure

```
.
├── src/arrears_risk_model/   # production code (config, schemas, data, features, models, train, predict, evaluate)
├── tests/                    # pytest suite, including synthetic fixtures
├── notebooks/                # exploratory analysis (EDA, clustering, modelling)
├── config/                   # YAML configuration files
├── data/                     # input data (gitignored — see data/README.md)
├── docs/                     # model card, architecture diagram
├── results/plots/            # plots generated by the analytical notebooks
├── models/                   # trained model artefacts (gitignored)
├── outputs/                  # predictions, priority lists (gitignored)
└── .github/workflows/        # CI
```

## Installation

```bash
uv sync
```

## Usage

The full pipeline is built up across subsequent commits. Once available:

```bash
# Train the models and write artefacts to models/{timestamp}/
uv run python -m arrears_risk_model.train

# Score new data and emit a ranked priority list
uv run python -m arrears_risk_model.predict
```

## Tests

```bash
uv run pytest
```
